# 営業日報システム テスト仕様書

## 1. テスト概要

### 1.1 目的
本書は営業日報システムの品質を担保するためのテスト仕様を定義する。

### 1.2 テスト範囲

| テスト種別 | 対象 | 概要 |
|-----------|------|------|
| 単体テスト | API、コンポーネント | 個々の機能の動作確認 |
| 結合テスト | API連携、画面遷移 | 機能間の連携確認 |
| システムテスト | システム全体 | エンドツーエンドの動作確認 |
| 非機能テスト | 性能、セキュリティ | 非機能要件の確認 |

### 1.3 テスト環境

| 環境 | 用途 | URL |
|------|------|-----|
| 開発環境 | 単体テスト | dev.example.com |
| ステージング環境 | 結合・システムテスト | stg.example.com |
| 本番環境 | リリース後確認 | app.example.com |
